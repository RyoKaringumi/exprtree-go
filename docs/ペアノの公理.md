# ペアノ公理

ペアの公理は、自然数に限定された公理系です。

数値として扱われるものは自然数(0を含む)で、小数点や負の数は含みません。

基礎的に、一項関数(一つの引数のみを受け取る関数)として、Sが定義されています。

Sは後者関数と呼ばれ、引数として受け取った自然数の次の自然数を返します。

例えば、1 = S(0)、2 = S(S(0))、3 = S(S(S(0)))のように表現されます。

## 公理1

```LaTeX
0 \in \mathbb{N}
```

意味: 0は自然数である。

## 公理2
```LaTeX
\forall n \in \mathbb{N},; S(n) \in \mathbb{N}
```

意味: 任意の自然数nに対して、その次の自然数S(n)も自然数である。

## 公理3
```LaTeX
\forall n \in \mathbb{N},; S(n) \neq 0
```

意味: 任意の自然数nに対して、その次の自然数S(n)は0ではない。

### 公理 4

```LaTeX
\forall a,b \in \mathbb{N},;
S(a)=S(b) \Rightarrow a=b
```

意味: 任意の自然数a,bに対して、S(a)がS(b)に等しいならば、aはbに等しい。

つまり、同じ自然数の次の自然数は一意に定まるし、その逆に異なる自然数の次の自然数が等しくなることはない(次の自然数が等しいなら、入力も等しいという式で示された)。

### 公理5 (数学的帰納法の原理)

```LaTeX
\bigl(0 \in A ;\land;
(\forall n\in A,; S(n)\in A)\bigr)
\Rightarrow
A=\mathbb{N}
```

意味: ある集合Aが0を含み、かつ任意の自然数nがAに含まれるならば、その次の自然数S(n)もAに含まれるならば、Aは自然数全体の集合である。
